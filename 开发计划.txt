一、项目概述

本项目旨在为企业建立一个英文产品展示网站，包括前台产品展示及后台管理系统。网站提供 “Home”, “Products”, “About”, “Contact” 四大导航，并支持后台产品数据的增删改查及询盘管理。

二、功能需求
1. 前台网站功能

导航栏：

Home
Products
About
Contact

Home 页面

企业简介
主要产品推荐（轮播/精选）

Products 页面

产品列表展示（图片、名称、简介）
点击单个产品进入详情页detail页
产品详情页（图片、详细描述、参数）
“Send Inquiry”功能（用户可提交询盘）

About 页面

公司介绍
企业文化、历史、资质证书等
Contact 页面

联系方式（电话、邮箱、地址、地图）
在线留言/询盘表单
响应式设计

手机、平板、电脑端自适应

2. 后台管理功能

管理员登录

产品管理

产品列表
产品新增
产品编辑
产品删除

询盘管理

显示用户发送的询盘
询盘详情查看
询盘处理/标记为已处理


三、技术架构建议
前端
 HTML + CSS3 + JavaScript
UI 框架： Bootstrap 或 tailwindcss

后端
cloudflare workers + d1 数据库
